## 使用方法

### 方法 1: 使用脚本（推荐）

```bash
export GITHUB_TOKEN=your_token_here
./update_github_repo.sh
```

### 方法 2: 手动添加

1. 访问仓库主页：https://github.com/vistone/netconnpool-rust
2. 点击仓库名称下方的 "⚙️" 图标（在 About 部分）
3. 在 "Topics" 字段中粘贴上述标签（每行一个或逗号分隔）
4. 点击 "Save changes"
